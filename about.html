How Starbyte Quest works

No server • No logins • No tracking • No external scripts

Starbyte Quest is a single static website you can host on GitHub Pages. Everything runs inside the child’s browser. That means it works on phones, tablets, laptops, desktops, and ultra-wide screens without installing anything.

What your child actually “codes”

Kids write (or build with blocks) very small commands like: say("Hi!"), moveRight(120), repeat(3, () => addStar()). These commands control a little hero in the Game Screen.

Safety model (why it’s safe to run kid code)

The Game Screen is an iframe sandbox (a mini isolated browser environment). Your child’s code runs inside that sandbox. The outer page only receives a small “state report” (like position and number of stars), and uses that to decide if the level is completed.

  • No network calls are required for the game to work.
  • No external images, fonts, or analytics.
  • No accounts. No uploads. No personal data.

How level checking works

After your child presses Run, the game sends back a simple state object: x, y, size, stars, etc. Each level has a small rule like “did the hero talk?” or “did stars >= 3?”.

Age packs (4–10)

  • Ages 4–5: block mode (drag/drop), talk/move/colour/stars.
  • Ages 6–7: beginner typing, introduces repeat and buttons.
  • Ages 7–8: recommended, introduces variables and functions.
  • Ages 9–10: adds if/else and more challenge tasks.

Block Mode (Ages 4–5)

New: Block Mode includes a friendly Coach bar with Guided Mode (highlights the next block) and optional Voice tips (Text-to-Speech). These are great for pre-readers.

In Ages 4–5, kids don’t need to type. They drag blocks into a simple list: “Say”, “Move”, “Change colour”, “Add stars”. Each block has tiny inputs (like a number or a colour). The app converts blocks into real code behind the scenes and runs it.

Tip: On touch devices (phones/tablets), kids can tap a block to add it (no dragging required).

Parent Mode

Parent Mode is protected by a simple local PIN. It lets you:

  • Jump to any level
  • Unlock all stickers
  • Reset progress
  • Edit level story/objective/hint text (locally)
  • Add unlimited custom levels with the Level Builder
  • View a dashboard of “skills used” (approx): loops, variables, functions, etc.

Note: Parent Mode is “kid-proofing”, not high-security. Everything is local on the device.

Stickers, albums, and mini-games

  • Each completed level gives 1⭐ (stored locally).
  • Stickers unlock as ⭐ increases. Sticker packs let kids choose a vibe.
  • Mini-games can award small ⭐ bonuses.
  • Hold Shift and click the Game Screen to place stickers on the stage.

Responsive design (phones → ultra-wide)

The layout uses a flexible grid: it’s 3 columns on big screens, 2 columns on medium screens, and 1 column on phones. The Game Screen and editor use viewport-based heights so they stay usable without scrolling forever.

Profiles (siblings)

Starbyte Quest supports multiple local profiles so siblings can each have their own stars, stickers, and progress. Profiles are stored on the device using browser storage.

World Map + Story Mode

Kids can choose a World (Candy, Unicorn, Space, Dino, Castle, Pirate, Neon). The world automatically picks a matching theme + sticker pack, and Story Mode shows a simple adventure line as they level up.

Daily Challenge

Each day the app offers a small coding challenge. Completing it awards bonus ⭐. This is local to the device.

Avatar Creator

Kids can choose a hero face, hats, and a pet. Some accessories unlock with ⭐.

Share a Creation

The Share feature exports blocks or code into a link (stored in the URL). It also shows a QR code so you can scan it with a phone camera. No server is required. Opening the link loads the creation on any device.

Mini-game Maker

The Maker lets kids build a mini-game with blocks and run it in the Game Screen.

Trophy Cabinet + Confetti

Trophies unlock automaticallyy from milestones (stars, badges, dailies). New trophies trigger a confetti celebration.

Installable (PWA)

Starbyte Quest supports offline use via a Service Worker and can be installed like an app (Add to Home Screen).

Language Packs

You can switch the interface language (English, Te Reo Māori, Español) from the header.

Narration

Turn on Narration to have the story and some tips read aloud using your device's Text-to-Speech.

Progress Report

Parents can print a weekly progress report (PDF) showing levels completed and skills practiced.

How to add more content

You have two options:

  • Easy: Use Parent Mode → Level Builder to create new levels on that device.
  • Permanent: Edit app.js and add levels to the pack functions.

Hosting on GitHub Pages

Put these files in your repo root: index.html, about.html, style.css, app.js. Then enable GitHub Pages from the main branch.

Troubleshooting

  • Nothing runs: press Run again or use Restore Starter / Clear Blocks.
  • Buttons A/B/C do nothing: the level may require wiring with onButton.
  • Sound doesn’t play: some browsers require a user tap first (toggle Sound and press Run).
  • Progress disappeared: clearing browser storage clears progress (it is stored locally).

Starbyte Quest is designed to be fun, safe, and easy to extend.
